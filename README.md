# CoderBuddy - AI Website Generator

CoderBuddy is an AI-powered website generation platform that creates complete websites from simple text prompts. Users can describe the type of website they want, and the system automatically generates the corresponding HTML, CSS, and JavaScript files along with a live preview.

The project was developed using FastAPI, Groq AI, LangChain, and modern frontend technologies. It streamlines the web development process by transforming natural language descriptions into functional and responsive websites within seconds.

### Key Features

* Generate complete websites using AI-powered prompts
* Create clean HTML, CSS, and JavaScript code automatically
* Live website preview before downloading
* Responsive and modern user interface
* Download generated website files instantly
* Error handling and validation for reliable generation
* Support for multiple AI models through Groq

### Technologies Used

* Python
* FastAPI
* Groq API
* LangChain
* Pydantic
* HTML5
* CSS3
* JavaScript

### How It Works

1. The user enters a description of the desired website.
2. The prompt is sent to the AI generation engine.
3. Groq AI generates structured website content.
4. The backend processes and validates the generated output.
5. HTML, CSS, and JavaScript files are created automatically.
6. Users can preview and download the generated website.
